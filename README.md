# 🏦 Bank Management System

A desktop-based Bank Management System developed using **Python** and **Tkinter**. This project demonstrates Python programming concepts including Object-Oriented Programming (OOP), GUI development, file handling, and JSON data storage.

---

## ✨ Features

- Create New Account
- Deposit Money
- Withdraw Money
- Search Account
- Delete Account
- View All Accounts
- View Total Bank Balance
- Transaction History
- Data stored permanently using JSON files

---

## 🛠 Tech Stack

- Python
- Tkinter
- JSON
- Object-Oriented Programming (OOP)
